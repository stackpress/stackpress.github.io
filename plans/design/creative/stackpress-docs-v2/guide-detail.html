    API usage guide

    Pages

    A page route is where an incoming request becomes an outgoing answer. The lesson starts with the route problem, then follows the smallest useful handler.

    What a page handler solves

    A browser asks for a path, and the app needs one focused place to decide what answer belongs there.

    Minimal handler

    server.get('/hello', ({ res }) => {
  res.html('<h1>Hello Stackpress</h1>');
});

    Follow the request

    1. Browser requests `/hello`The route owns the URL match.
    2. Handler prepares the answerThe handler reads input and chooses output.
    3. Response sends HTMLThe result is visible immediately.

    Check it

    The closing section tells the reader what changed, how to verify the page, and which idea should be clear before the next guide.
