# Stackpress Docs Creative Draft V1

This is a static creative design-review draft based on the approved wireframes in `packages/www/design/wireframes/stackpress-docs-v1`.

## Stage

Creative phase, Round 3.

## Screens Included

- `index.html` for the homepage / docs entry direction
- `concepts-index.html` for the concept shelf / landing page
- `concepts.html` for the article reading surface
- `guides.html` for the procedural guide overview
- `guide-detail.html` for a task-focused tutorial page
- `api.html` for the reference index without search
- `api-detail.html` for a lookup-focused reference detail page
- `assets/logo.png` as a local copy of the Stackpress wordmark for draft preview

## Visual Direction

- Docs-first product-documentation hybrid.
- Stackpress blue is the primary action and orientation color.
- Aqua/mint is a secondary technical accent for framework model surfaces.
- Reading pages are calmer than the homepage.
- Code blocks, path steps, and article navigation are treated as trust surfaces.
- Dark mode is available through the sun/moon header control.
- Mobile navigation separates the global site menu from page-local docs controls.

## Review Notes

- This is not production implementation.
- Links are static and included for visual review.
- Search has intentionally been removed from the draft direction.
- Mobile controls are simulated but clickable enough for review.
- Round 3 is the final polish pass before Creative phase approval or another feedback revision.
