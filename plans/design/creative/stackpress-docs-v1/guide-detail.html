Hello World Route

A task-focused page should be linear, with a clear outcome, short explanations, and code blocks that are easy to scan.

Outcome A local route returns visible output through the Stackpress server.

Create the route

Keep the first step small. The reader should know exactly where the file lives and what result to expect.

server.import.get('/hello', () => import('./pages/hello.js'));
server.view.get('/hello', '@/plugins/app/views/hello');

Run it

Use a short verification step before introducing more framework layers.

yarn dev
open http://127.0.0.1:3000/hello
