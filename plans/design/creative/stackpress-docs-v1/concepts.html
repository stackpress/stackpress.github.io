Overview

A calm reading page for conceptual docs. The page prioritizes fast scanning, readable prose, and persistent orientation.

Start here: Stackpress is a schema-first application framework that combines idea files, config, plugins, generated code, and server-rendered views.

Quick Start Mental Model

Author the source of truth, run generation, push schema changes into the database, then run and inspect the app.

schema.idea -> generate -> push -> serve

Core Concepts

  • schema.idea is the main authoring surface.
  • config/*.ts controls orchestration and runtime setup.
  • Plugins own app-specific behavior and extension points.
  • Generated output is disposable and inspectable.
