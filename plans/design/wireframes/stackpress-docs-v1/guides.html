Getting Started

A practical guide page should keep developers moving through visible milestones. It can be more procedural than Concepts, but still needs enough context to explain why each step exists.

Recommended path: Start with the numbered tutorial track, then use supporting guides for project structure, generation, views, plugins, and debugging.

Tutorial Track

  1. Hello World RouteProves the smallest route works.
  2. First React PageAdds the view layer.
  3. First Project ShapeIntroduces config and plugins.
  4. First Schema GenerationIntroduces idea files and generated output.
